# Frontend — Liga de Baloncesto TPO

SPA en **React 19 + Vite** con Tailwind CSS v4 y shadcn/ui. Portal público (standings, playoffs, palmarés, equipos, jugadores) + panel de administración protegido con JWT.

```bash
npm install
npm run dev     # http://localhost:5173 — requiere el backend en localhost:3000
```
